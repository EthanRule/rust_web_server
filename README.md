# Rust-Web-Server

Rust Web Server

## Support

- HTTP/2
- IPV4, IPV6
- GET, POST

## Commands

### Run Code
```cargo run```  

### Send Client -> Server Requests
```curl.exe -X POST http://localhost:3000/echo/reversed -d "Some data"```  
```curl.exe http://localhost:3000/```  

###
Web server is a proxy between front and backend. Examples are Node and Bun
